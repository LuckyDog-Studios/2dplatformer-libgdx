# Platformer Prototype

**Platformer Prototype** is a barebones, unfinished platformer prototype built with LibGDX.

---

## Tools:
- LibGDX
- Java
- Aseprite (art)

---

## How to Run

**WINDOWS:**
1. Download the zip file in [Executable Folder](/platformer/dist)
2. Extract files
3. Launch the executable and start playing

---

## Controls

- **W / A / S / D** — Move
- **Space** — Jump
- **Left-Click** — Slash

---

## Note:
This is an unfinished prototype.
